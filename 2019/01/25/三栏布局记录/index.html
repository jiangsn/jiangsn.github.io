使用float布局

左右固定100px ，中间内容自适应， 使用float布局 html的格式只能怪居中的要放在浮动的后面

<body>
    <div class="left"></div>
    <div class="right"></div>
    <div class="center">
        <div>
            <p>居中的内容居中的内容居中的内容居中居中的内容居中的内容居中的内容居中的内容的内容</p>
        </div>
    </div>
</body>

    body{
      width:100%;
      height:100vh;
    overflow: hidden;
    }
.left,.right{
    width:100px;
    height:100%;
    }
.left{
    float:left;
    background-color:blue;
    }
.right{
    float:right;
    background-color:green;
    }
.center{
      height:100%;
    background-color:gold;
    }

使用grid布局

    * {
  margin: 0;
  padding: 0;
}
body {
  display: grid;
  width: 100%;
  grid-template-columns: 100px auto 100px;  // 每行的占的宽度
  grid-template-rows: 100vh;     // 行高
}

.left {
  background-color: blue;
}
.right {
  background-color: green;
}
.center {
  background-color: gold;
}
