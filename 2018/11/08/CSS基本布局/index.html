CSS基本布局

2018/11/08 | CSS

左右布局

在一个div中左右将两个div左右排放

.clearfix::after{
    content: "";
    display: block;
    clear: both;
}
.parent{
    border:1px solid green;
}
.left{
    width: 30%;
    height:100px;
    background-color: gold;
    border:1px solid green;
    float:left;
}
.right{
    width: 70%;
    height:100px;
    background-color: green;
    border:1px solid gold;
    float:left; 
}

左中右布局

  • 使用position：absolute，定位左右，中间margin撑开
  • 作用使用float,使得左右脱离文档流 ，中间部分用margin撑开

水平居中

  • inline元素水平居中 在父类中使用text-align：center；
  • block元素水平居中使用
.center-me {
  margin: 0 auto;
}

垂直居中

  • 单个inline元素垂直居中
    1.使用padding-top和padding-bottom相等
    2.如果因为某些原因不能使用padding属性
  • block元素
    1.不知道高度
.parent {
  position: relative;
}
.child {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
}

2.flex布局

.parent {
  display: flex;
  flex-direction: column;
  justify-content: center;
}

水平垂直居中

  • 固定宽高
    使用负值margin等于宽的和高的一半 ，然后使用padding：50% 。 具有很好的夸浏览器支持。
.box {
    width:200px;
    height: 200px;
    border: 1px black solid;
    position:relative;
}
.one{
   background:green;
   width:50px;
   height: 50px;
   margin: -25px 0px 0px -25px;
   position:absolute;
   top:50%;
   left:50%;
}
  • 不定宽高
    在支持 transform: translate(-50%, -50%)属性
main {
  position: relative;
}
div {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
  • 使用flex布局
.parent {
  display: flex;
  justify-content: center;
  align-items: center;
}
  • 使用grid布局
    在一个元素时，有很好的效果。
body, html {
  height: 100%;
  display: grid;
}
span { /* thing to center */
  margin: auto;
}
