HTML常用标签
Publish: 2018/10/29   
  • HTML

iframe

iframe标签用户嵌套页面，可以a标签搭配使用。iframe标签的name属性的值和a标签的target的值相同。

<iframe src="https://www.baidu.com" name="xxx"></iframe>
<a href="#" target ="xxx">链接内容将会在iframe内展现</a>

a

跳转页面（HTTP GET 请求）

  • href: 包含超链接指向的URL或URL片段
  • target: 该属性指定在何处显示链接的资源.(与iframe的name值相同时，链接的资源可展示在iframe中)
tip：  伪协议的使用。什么时候用我也不知道。但可以这样写。

form

跳转页面（HTTP POST 请求)用来向web服务器提交信息。

  • action 将表单数据提交到服务器。可以被button、input属性重载。
  • method 提交的方法。
  • name html5中 当文档中有多个form时，name值必须唯一且不能是一个空字符串。
  • target 来指示在提交表单之后，在哪里显示收到的回复，取值如下：
    – _self： 在当前frame中打开。
    _blank： 以新的窗口加载返回值。
    _parent： 在父级的frame中，如果没有同_self表现一样。
    _top：在当前文档的最高级内加载返回值，如果没有父级，和_self的行为一致。
    iframename：返回值在指定frame中加载。
<form action="">
  <label for="GET-name">Name:</label>
  <input id="GET-name" type="text" name="name">
  <input type="submit" value="Save">
</form>

input

input的type有很多详细可下MDN.

https://developer.mozilla.org/zh-CN/docs/Web/HTML/Element/Input

HTML5增加了许多有用取值：

  • date ，datetime，datetime-local，month，number，range，search，tel，time，url，week
  • accept 如果该元素的 type 属性的值是file,则该属性表明了服务器端可接受的文件类型；否则它将被忽略。
tip：空元素即不可能存在子节点（内嵌元素或者元素内的文本）的元素。

button

name / value
  • 当button有name属性时，提交表单会把value的值一同提交给服务器。

#####　 type

  • submit 默认值，位于表单内，将表单内容提交给服务器
  • reset 重置
  • button 与元素的事件绑定，当事件发生时触发。
<p><button>Default button</button></p>

<p><button disabled>Disabled button</button></p>

<p>
  <button name="submit" type="submit" value="submit-true">
    Form submit button
  </button>
</p>

<p><button accesskey="a">Button with <u>A</u>ccesskey</button></p>

<p><button class="styled">Fancy styled button</button></p>
