原型与原型链

2018/12/09 | JS

全局对象

  • 浏览器内置对象
  • window
  • window属性分类： ECMASript规定（parseInt ，parseFloat，Number ，etc） ，浏览器自己实现的（alert，prompt，confirm，document，etc）。

原型和原型链

  • MDN的描述：每个对象拥有一个原型对象。对象以其原型为模板、从原型继承方法和属性。原型对象也可能拥有原型，并从中继承方法和属性，一层一层、以此类推。这种关系常被称为原型链 (prototype chain)。

  • tip： 对象的原型是每个实例上都有的属性。

  • 构造函数的prototype属性，后者是构造函数的属性。

每个函数都有一个特殊的属性叫作原型（prototype),

  • proto : 大多数现代浏览器还是提供了一个直接访问的原型的属性。
  • prototype：原型链中的“连接”被定义在一个内部属性中，在 JavaScript 语言标准中表示。
  • constructor：用于构造此实例对象的构造函数。
